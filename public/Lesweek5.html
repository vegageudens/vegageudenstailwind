Weekplannning

5de week

In week 5 zijn we zelf aan de slag gegaan met HTML en CSS. We moesten dus van een site een bepaalde website dowloaden die we dan zelf moesten oppimpen. We moesten dus eens de achtergrond van kleur veranderen en het font. We hebben dus een nieuw font moeten plaatsen in onze CSS. Als je dus in Google fonts verder klikt kom je HTML en CSS code tegen die je dus zelf kan plaatsen in je eigen code. Hier heb ik dan ook een beetje mee gespeeld tot ik een goed lettertype vond die bij de stijl van de site paste. In de deze week hebben we ook meer uitleg gekregen over HTML en CSS en wat het nu uiteindelijk allemaal inhoud.
