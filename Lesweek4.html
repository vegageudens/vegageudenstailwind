My first page

4de week

In de 4de week heb ik van mijn treinborden een screenshot genomen en geplaats in chat chpt. Ik heb toen gevraagd om deze om te zetten naar code die ik dan daarna kon plakken in Visual Studio Code. Hier en daar vond ik de code toch niet zo goed geschreven. Ik gevraagd om paar codes te vervangen waar ik zelf het meeste van begrijp bv sommige code te veranderen naar div id en div class. Daarna heb ik de aangepaste code terug geplakt in Visual Studio Code. Toen ik naar mijn pagina was gaan kijken in mijn browser zag het er toch niet helemaal hetzelfde, dus moest ik zelf nog een beetje aan prutsen zodat het er meer op zou lijken, maar het is nog steeds niet volledig wat ik wil.
